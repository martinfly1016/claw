# OpenClaw-Navi.jp

日本最大の OpenClaw 攻略ポータルサイト。

## 開発スタック
- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Deployment:** Vercel / GitHub Pages

## 公開予定コンテンツ
- `使い方`: 初心者向け導入ガイド
- `スキル一覧`: 厳選 Skill データベース
- `VPS推薦`: 24時間稼働のためのサーバー選定
- `比較`: OpenClaw vs 云端Agent
