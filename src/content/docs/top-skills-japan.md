# 日本市場向け OpenClaw 推奨スキル Top 10

日本国内のビジネス環境において、OpenClaw のポテンシャルを最大限に引き出すための必装スキルを 10 個厳選しました。
※ 本ページは `top-skills-japan` と `clawhub-top10` を統合し、最新情報に更新したものです。

---

## 1. agent-browser (Web 操作自動化)
- **用途**: 競合サイトの定期巡回、管理画面への自動入力。
- **理由**: 日本の商習慣に多い「ブラウザベースの反復作業」を完全に自動化できます。

## 2. tavily-search (高精度外部検索)
- **用途**: 最新の技術トレンド調査、ニュース収集。
- **理由**: Google 検索よりも「根拠のある情報の抽出」に特化しており、レポート作成が捗ります。

## 3. summarize (多機能要約)
- **用途**: 長文メール、議事録、ログの要点整理。
- **理由**: 情報過多な現代において、意思決定に必要な 20% の情報を即座に抽出します。

## 4. github (開発運用連携)
- **用途**: Issue の自動要約、PR の作成支援、CI 失敗分析。
- **理由**: 日本のテックチームにおける GitHub 依存度は高く、自動化の恩恵が最も大きいです。

## 5. gog (Google Workspace 連携)
- **用途**: Gmail 下書き作成、カレンダー調整、ドライブ整理。
- **理由**: 組織の基本インフラである Google ツールとのシームレスな連携を実現します。

## 6. nano-pdf (PDF 高速処理)
- **用途**: 契約書の要点抽出、論文の比較分析。
- **理由**: 日本企業に根強く残る PDF 資産を、再利用可能なテキストデータに変換します。

## 7. self-improving-agent (改善ループ)
- **用途**: プロンプトの自己修正、成功率の向上。
- **理由**: 使えば使うほど賢くなる「自律型エージェント」の真髄を体験できます。

## 8. message (通知連携)
- **用途**: Discord, Slack, Telegram への完了通知。
- **理由**: エージェントの動作状況をリアルタイムで把握し、チームで共有するために必須です。

## 9. healthcheck (インフラ監視)
- **用途**: VPS や自社環境のセキュリティ監視、脆弱性診断。
- **理由**: 自律エージェントを動かす環境そのものの安全性を AI に守らせます。

## 10. weather (実務連携例)
- **用途**: 出張計画の補助、配送スケジュールの調整。
- **理由**: API 連携の入門としても最適で、かつ実用性の高い外部データ活用の見本です。

---

## 導入の進め方
まずは **1〜3** を導入して「情報の取得と要約」を AI に任せることから始め、徐々に **4〜7** の高度な自動化へ移行することを推奨します。
