# OpenClaw WeChat 連携トラブルシューティング

WeChat連携で頻出する障害を、即時復旧できる形でまとめました。

## 1. 認証失敗
- 原因: キー期限切れ / 設定ミス
- 対処: キー更新、設定再読込、接続確認

## 2. 通知が遅延
- 原因: キュー詰まり / APIレート制限
- 対処: リトライ間隔調整、優先度キュー導入

## 3. 送信先ミス
- 原因: ルーティング表未更新
- 対処: 宛先マップを単一ソース化

## 4. 障害時に連投
- 原因: 停止条件なし
- 対処: エラー閾値で自動停止

## 推奨運用
- 日次: エラーログ確認
- 週次: 宛先・通知品質レビュー
- 月次: 鍵ローテーション
