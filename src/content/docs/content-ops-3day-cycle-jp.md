# Claw-Navi コンテンツ運用（3日サイクル）

## 目的
SEMrushキーワード優先度とXシグナルを統合し、継続的に高品質ページを公開する。

## サイクル
### Day 1: シグナル収集
- x-content-scout を実行
- 候補を A/B/C に分類（60% SEMrush + 40% X）

### Day 2: 制作
- Aランクから1〜2本作成
- 各ページに FAQ / 外部リンク / metadata を付与

### Day 3: 公開とQA
- 本番反映
- 可用性・検索・構造化データ・sitemap を確認
- 結果を次サイクルへ反映

## KPI
- 新規インデックス数
- 主要KW順位
- docs回遊率
- CTAクリック率
