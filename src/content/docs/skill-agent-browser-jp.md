# agent-browser 運用ガイド（実務版）

agent-browser は、ブラウザ操作を自動化して
「調査・入力・確認」タスクを高速化するためのスキルです。

## 1. 使いどころ
- 定期的な情報収集
- 管理画面の反復操作
- フォーム入力や確認作業の自動化

## 2. 導入前チェック
- [ ] 操作対象サイトの権限確認
- [ ] 自動化対象ステップの明文化
- [ ] 失敗時の停止条件を定義

## 3. 実装ステップ
### Step 1: スキル導入
- agent-browser を導入し、接続環境を確認

### Step 2: 操作手順の分解
- 1タスクを「遷移→入力→確認」に分割
- 各ステップに成功条件を設定

### Step 3: リトライ戦略
- タイムアウト時の再試行回数を固定
- 失敗ログを残し、手動復旧導線を準備

## 4. 失敗しやすいポイント
1. 動的ページで要素参照がズレる
2. ログイン状態が切れて失敗する
3. 例外ケースが定義されていない

## 5. 安全運用ルール
- 破壊的操作（削除・公開）は人間承認
- 本番操作前に検証環境で試験
- 実行履歴を保存し監査可能にする

## 6. KPI
- 手動作業削減時間
- 自動化成功率
- 障害時復旧時間（MTTR）
