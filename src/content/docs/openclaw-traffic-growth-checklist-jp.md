# Claw-Navi 流入成長チェックリスト（公開後7/14/30日）

公開した記事を「出して終わり」にしないための、実務チェックリストです。

## 0-24時間（公開直後）
- [ ] サイト表示崩れがない
- [ ] タイトル / description / canonical が正しい
- [ ] sitemap に反映されている
- [ ] 内部リンク（関連3本以上）を設置

## 7日チェック
- [ ] Search Console でインデックス確認
- [ ] 表示回数が出ているか
- [ ] 想定キーワードで初期順位が付いているか

## 14日チェック
- [ ] クリック率（CTR）を確認
- [ ] タイトル最適化（必要なら差し替え）
- [ ] FAQ追記でSERP適合度を上げる

## 30日チェック
- [ ] 流入貢献ページ上位を抽出
- [ ] 低成果ページを再設計（H2/導入/内部リンク）
- [ ] 次サイクルの優先語を更新

## 高速改善ループ
1. 低CTR記事 → タイトル改善
2. 低滞在記事 → 導入文と結論位置を改善
3. 低順位記事 → 関連クラスター記事を追加

## 運用ルール
- 指標を見ずに記事本数だけ増やさない
- 3日ごとに必ず1回は既存記事を改善する
- 内部リンク更新をコンテンツ制作と同時に実施
